Direct Line Group

I was employed by Direct Line Group to work on the front-end of a new web app that is due to be rolled out. The new app is a single-page application built with AngularJS, and allows customers to be able to get an instant quote quickly and easily.

This role included testing and debugging the app, adding and amending Angular content, and making html and css changes where necessary. The team had created an adaptive design approach, so I was working on separate mobile and desktop versions.

I also helped create, debug, and test mobile versions of the other differently branded incarnations of the app; Churchill, Privilege and Green flag.
